# Java vs Bedrock Edition, Edited-Updated

After purchasing your Minecraft server with HostByte, you'll need to access your **HostByte Game Panel** to manage everything. This comprehensive guide shows you how to log in, navigate the panel, and use all the essential features to control your server. ✅

***

### ⛏️ Step 1: Access Your HostByte Control Panel

Your HostByte control panel is your command center for managing your Minecraft server. Here's how to access it:

1. ➡️ **Check Your Welcome Email** - Look for login credentials sent to your email
2. ➡️ **Visit the Panel URL** - Usually `https://game.hostbyte.com` or similar
3. ➡️ **Enter Your Credentials** - Use the username and password from your email
4. ➡️ **Click Login** - Access your server dashboard

***

### ⚙️ Step 2: Navigate Your Server Dashboard

Once logged in, you'll see your server dashboard with these key sections:

* ✔️ **Server Overview** - Current status and basic information
* ✔️ **Power Controls** - Start, stop, restart your server
* ✔️ **Console Access** - Real-time server console
* ✔️ **File Manager** - Upload and manage server files
* ✔️ **Settings** - Configure server options
* ✔️ **Backups** - Manage server backups

***

### ⚙️ Step 3: Power Management

Control your server's power state using the HostByte panel:

* ✅ **Start Server** - Power on your Minecraft server
* ❌ **Stop Server** - Safely shut down the server
* ⚙️ **Restart Server** - Quick restart for updates
* ⚠️ **Force Stop** - Emergency shutdown (use carefully)

***

### ⚙️ Step 4: Console Access

The console is your direct connection to your server:

* ✔️ **Real-time Logs** - See server activity as it happens
* ✔️ **Command Input** - Send commands directly to your server
* ✔️ **Error Monitoring** - Watch for issues and errors
* ✔️ **Player Activity** - Monitor player connections and actions
