# How to Schedule Server Backups or Restarts

Automating your server maintenance tasks can save time and ensure your Minecraft server runs smoothly. This guide shows you how to set up scheduled backups and restarts using HostByte's automation features to keep your server healthy and your data safe. ⚙️

---

## ⚙️ Understanding Scheduled Tasks

Scheduled tasks help maintain your server automatically:

- ✅ **Regular Backups** - Protect your world data
- ✅ **Automatic Restarts** - Prevent memory leaks and lag
- ✅ **Maintenance Windows** - Scheduled downtime for updates
- ✅ **Performance Optimization** - Keep server running smoothly
- ✅ **Data Protection** - Ensure no data loss

### ⚙️ Benefits of Scheduling

**Why schedule server tasks:**

- ⚠️ **Consistency** - Regular maintenance without manual intervention
- ⚠️ **Reliability** - Prevent server crashes and data loss
- ⚠️ **Convenience** - Set it and forget it approach
- ⚠️ **Performance** - Maintain optimal server performance

---

## ⚙️ Step 1: Access Scheduled Tasks

Navigate to the scheduling section in your HostByte panel:

### ⚙️ Find Scheduling Options

1. ➡️ Log into your HostByte control panel
2. ➡️ Navigate to your Minecraft server
3. ➡️ Look for "Schedules" or "Automation" in the sidebar
4. ➡️ Click on "Scheduled Tasks" or "Automation"

### ⚙️ Schedule Management Interface

- ✅ **Task List** - View all scheduled tasks
- ✅ **Create New** - Add new scheduled tasks
- ✅ **Edit Existing** - Modify current schedules
- ✅ **Task History** - View past executions
