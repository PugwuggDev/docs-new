# How to Start, Stop, or Restart Your Server

Controlling your Minecraft server's power state is one of the most fundamental management tasks. This guide shows you how to safely start, stop, and restart your server using HostByte's control panel, ensuring smooth operation and data protection. ⚙️

---

## ⚙️ Understanding Server Power States

Your Minecraft server can be in different power states:

- ✅ **Online** - Server is running and accepting players
- ✅ **Starting** - Server is booting up
- ✅ **Stopping** - Server is shutting down
- ✅ **Offline** - Server is powered off
- ✅ **Installing** - Server is being set up

### ⚙️ Power Management Options

**Available control actions:**

- ⚠️ **Start** - Power on the server
- ⚠️ **Stop** - Safely shut down the server
- ⚠️ **Restart** - Quick restart for updates
- ⚠️ **Force Stop** - Emergency shutdown (use carefully)

---

## ⚙️ Step 1: Access Power Controls

Navigate to the power management section in your HostByte panel:

### ⚙️ Find Power Controls

1. ➡️ Log into your HostByte control panel
2. ➡️ Navigate to your Minecraft server
3. ➡️ Look for power control buttons on the dashboard
4. ➡️ Locate the server status indicator

### ⚙️ Power Control Interface

- ✅ **Status Display** - Current server state
- ✅ **Power Buttons** - Start, stop, restart controls
- ✅ **Progress Indicators** - Show operation progress
- ✅ **Status Messages** - Inform about current state
