# Creating Subusers for Server Management

Managing a Minecraft server can be a team effort. With HostByte's subuser system, you can grant specific permissions to trusted individuals without giving them full access to your account. This guide shows you how to create and manage subusers for effective server administration. ⚙️

---

## ⚙️ Understanding Subusers

Subusers provide controlled access to your server management:

- ✅ **Limited Permissions** - Grant specific access levels
- ✅ **Secure Management** - No full account access
- ✅ **Team Collaboration** - Multiple administrators
- ✅ **Activity Tracking** - Monitor subuser actions
- ✅ **Easy Removal** - Revoke access when needed

### ⚙️ Common Use Cases

**When to create subusers:**

- ⚠️ **Server Administrators** - Trusted staff members
- ⚠️ **Build Teams** - Creative team members
- ⚠️ **Moderators** - Player management staff
- ⚠️ **Technical Support** - Help with server issues

---

## ⚙️ Step 1: Access User Management

Start by accessing the subuser management section:

### ⚙️ Navigate to Subusers

1. ➡️ Log into your HostByte control panel
2. ➡️ Navigate to your Minecraft server
3. ➡️ Look for "Users" or "Subusers" in the sidebar
4. ➡️ Click on "Manage Subusers"

### ⚙️ Panel Navigation

- ✅ **Server Dashboard** - Access from main server page
- ✅ **Settings Menu** - Find in server settings
- ✅ **User Management** - Dedicated subuser section
- ✅ **Account Settings** - Access from account menu
