# Using the Console to Run Minecraft Commands

The console is your direct connection to your Minecraft server, allowing you to execute commands, monitor server activity, and troubleshoot issues in real-time. This guide shows you how to effectively use the HostByte console to manage your server and run Minecraft commands. ⚙️

---

## ⚙️ Understanding the Console

The console provides direct access to your server's command line:

- ✅ **Real-time Output** - See server activity as it happens
- ✅ **Command Execution** - Run Minecraft commands directly
- ✅ **Error Monitoring** - Watch for issues and errors
- ✅ **Player Activity** - Monitor player connections and actions
- ✅ **Server Logs** - View detailed server information

### ⚙️ Console Features

**Key console capabilities:**

- ⚠️ **Command Input** - Send commands to the server
- ⚠️ **Auto-scroll** - Follow latest activity automatically
- ⚠️ **Copy/Paste** - Easy command sharing and logging
- ⚠️ **Search Function** - Find specific log entries

---

## ⚙️ Step 1: Access the Console

Navigate to the console section in your HostByte panel:

### ⚙️ Find Console Access

1. ➡️ Log into your HostByte control panel
2. ➡️ Navigate to your Minecraft server
3. ➡️ Look for "Console" in the sidebar menu
4. ➡️ Click on "Console" to open the interface

### ⚙️ Console Interface

- ✅ **Output Window** - Real-time server output display
- ✅ **Command Input** - Text field for entering commands
- ✅ **Send Button** - Execute commands
- ✅ **Clear Button** - Clear console output
