# My Mods/Plugins Aren’t Working – What To Do

When your mods or plugins stop working, it can be frustrating and confusing. This comprehensive guide will help you diagnose and fix common issues with Minecraft mods and plugins, getting your server back to full functionality. ⚙️

---

## ⚠️ Understanding Mod/Plugin Issues

Mods and plugins can fail for various reasons:

- ❌ **Version Incompatibility** - Wrong Minecraft or mod loader version
- ❌ **Missing Dependencies** - Required mods or libraries not installed
- ❌ **Conflicting Mods** - Mods that interfere with each other
- ❌ **Corrupted Files** - Damaged or incomplete downloads
- ❌ **Configuration Errors** - Incorrect settings or permissions

### ⚠️ Common Error Types

**Typical mod/plugin problems:**

- ⚠️ **Plugin failed to load** - Startup errors
- ⚠️ **Mod crashes on startup** - Initialization failures
- ⚠️ **Features not working** - Commands or functions disabled
- ⚠️ **Server won't start** - Critical mod/plugin errors

---

## ⚙️ Step 1: Check Server Logs

Start by examining your server logs for error messages:

### ⚙️ Access Server Console

1. ➡️ Open your HostByte control panel
2. ➡️ Navigate to your server console
3. ➡️ Look for error messages during startup
4. ➡️ Note any specific plugin/mod errors

### ⚙️ Common Error Messages

- ⚠️ **"Plugin failed to load"** - Check version compatibility
- ⚠️ **"NoClassDefFoundError"** - Missing dependencies
- ⚠️ **"Incompatible version"** - Version mismatch
- ⚠️ **"Plugin disabled"** - Configuration issues
